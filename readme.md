## Contenidos:
- Vue & Vuex 💚
- Firebase: 
    - Storage 🌉
    - Firestore 📚
    - Authentication 🔒
    - Cloud Functions ⛅️
- Procesamiento de imágenes (Cloud Functions + Storage) 🌐
- Buscador con Algolia / Elasticsearch 🔍
- Open Graph (metadatos) 🔣
- Hosting & Deploy con Netlify ⛓
- Testing (Unit testing, E2E testing) ✅
